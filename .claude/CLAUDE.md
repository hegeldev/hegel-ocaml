# Hegel SDK for ocaml

## Build Commands

```bash
just setup   # Install dependencies and hegel binary
just test    # Run tests
just format  # Auto-format code
just lint    # Check formatting + linting
just docs    # Build API documentation
just check   # Run lint + docs + test
```

## Overview

This is a Hegel property-based testing SDK for ocaml.
Stage 1 will fill in tooling choices and project conventions.
