Module Hegel.Protocol

Binary wire protocol for packet serialization and deserialization.

Wire protocol for the Hegel SDK.

This module defines the binary packet format used for communication between the Hegel SDK and the Hegel server. Messages are serialized as packets with a 20-byte header, a CBOR-encoded payload, and a terminator byte.

The header consists of 5 big-endian unsigned 32-bit integers:

  • Magic number (0x4845474C, "HEGL")
  • CRC32 checksum
  • Channel ID
  • Message ID (high bit = reply flag)
  • Payload length
val magic : int32

Magic number identifying Hegel packets ("HEGL" in hex).

val header_size : int

Format string for the 20-byte header: 5 big-endian uint32 values.

val terminator : int

Terminator byte appended after the payload.

val reply_bit : int32

Bit flag indicating a reply message. When set in the message ID field, the message is a response to a previous request.

val close_channel_message_id : int32

Special message ID used when closing a channel.

val close_channel_payload : string

Special payload byte sent when closing a channel. Chosen to be invalid CBOR per RFC 8949 (reserved tag byte 0xFE).

type packet = {
  1. channel_id : int32;
    (*

    The logical channel this packet belongs to.

    *)
  2. message_id : int32;
    (*

    Identifier for request/response correlation.

    *)
  3. is_reply : bool;
    (*

    Whether this is a reply to a previous request.

    *)
  4. payload : string;
    (*

    The raw payload bytes (typically CBOR-encoded).

    *)
}

A single message in the wire protocol.

val equal_packet : packet -> packet -> bool

equal_packet a b returns true if packets a and b have identical fields.

val pp_packet : Stdlib.Format.formatter -> packet -> unit

pp_packet fmt p pretty-prints packet p to formatter fmt.

val compute_crc32 : Stdlib.String.t -> int32

compute_crc32 data computes the CRC32 checksum of data and returns it as an int32.

val pack_uint32_be : bytes -> int -> int32 -> unit

pack_uint32_be buf offset value writes value as a big-endian unsigned 32-bit integer at offset in buf.

val unpack_uint32_be : bytes -> int -> int32

unpack_uint32_be data offset reads a big-endian unsigned 32-bit integer from data at offset.

exception Partial_packet of string

Exception raised when the connection is closed partway through reading a packet.

exception Connection_closed of string

Exception raised when the connection is closed while reading data and some bytes have already been received.

val recv_exact : Unix.file_descr -> int -> bytes

recv_exact sock n reads exactly n bytes from Unix file descriptor sock. Raises Partial_packet if the connection closes with no data read, or Connection_closed if it closes after partial data.

val read_packet : Unix.file_descr -> packet

read_packet sock reads and parses a single packet from Unix file descriptor sock.

Raises Failure if the magic number is invalid, the terminator byte is invalid, or the CRC32 checksum does not match.

val write_packet : Unix.file_descr -> packet -> unit

write_packet sock packet serializes and writes packet to Unix file descriptor sock.
